  • Auf der Übersichtsseite besteht die Möglichkeit über die sich rechts befindende Buttons (Kalender und A-Z Abbildung) die Sortierung der Rezepte zu ändern. (Aktiver Button=schwarz, inaktive Button=leicht transparent)
    Möchte man ein Rezept löschen, klickt man auf das Herz. Daraufhin erscheint eine Meldung, ob das Rezept wirklich entfernt werden soll. Wird diese Meldung mit “OK” bestätigt, wird das Rezept aus der Übersicht entfernt.
    Möchte man das Rezept mit genaueren Informationen sehen muss dafür auf den Titel geklickt werden.
    Eigene Rezepte können durch anklicken des "Rezept hinzufügen" Button eingepflegt werden.
  • Auch hier besteht die Möglichkeit das Rezept zu löschen, indem man auf den Mülleimer klickt. Nach dem Anklicken muss ebenfalls die Meldung mit “OK” bestätigt werden.
    Eigene Rezepte können durch betätigen des Stift Buttons überarbeitet werden.
    Rezepte die aus der API abgespeichert wurden kann man sich über den "Zum Rezept" Button genauer anschauen.
